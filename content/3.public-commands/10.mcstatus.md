# Minecraft Status

"Minecraft server IP status" allows users to check the online or offline status of a Minecraft server using its IP address. By entering the server's IP address, users can quickly determine whether the server is online and operational, ensuring they don't waste time trying to connect to an inactive or temporarily unavailable server.

::callout
#summary
Mcstatus Command:

#content
/mcstatus >server<
::

::callout
#summary
Example:

#content
/mcstatus server:0.0.0.0
::

::list{type="info"}
- Server = IP of the Minecraft Server.
::
