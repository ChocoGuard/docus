# Purge

The Purge command allows moderators or administrators to swiftly delete a specific number of messages from a channel. (99/100 Messages MAX)

::callout
#summary
Purge Command:

#content
/purge >amount< >target<
::

::callout
#summary
Example:

#content
/purge amount:50 (99/100 Messages MAX) target:@Test1234
::
